Exercises 6 and 7:

Calculate the sample mean for samples of 10, 20, 50, 100, 200, 500, and 1000. Show how to do this using both a standard 'for' loop as well as the 'apply' function.

First, we're going to define a custom function to calculate the mean, SD, and 95% CI for a random sample of size N from a specified distribution.

(We will need the 95% CI in Exercise 8, and it is most efficient to calculate everything in a single function)

In [10]: 
%%R
# Define a new function called genSampleMean with two arguments, sample_size and distribution.
genSamples = function(sample_size, distribution) {
    # Take a sample of the specified size from the specified distribution.
    sample = sample(population, size=sample_size, replace=FALSE)
    # Get the mean of our sample.
    sample_mean = mean(sample)
    # Get the SD for our sample.
    sample_sd = sd(sample)
    # Calculate the 95% confidence interval for our sample mean.
    sample_error = (qnorm(0.975)*sample_sd)/(sqrt(sample_size))
    sample_ci_ub = sample_mean + sample_error
    sample_ci_lb = sample_mean - sample_error
    # Group the mean, SD, and error into a vector.
    sample_stats = c(sample_size, sample_mean, sample_sd, sample_error, sample_ci_ub, sample_ci_lb)
    # Output our vector of statistics.
    return(sample_stats)
}

Loops in R are slow, so let's use the apply function.

In [11]: 
%%R
# Create a vector containing all of the different sample sizes we want.
sample_sizes = cbind(10, 20, 50, 100, 200, 500, 1000)
# Generate statistics for each of sample sizes in our vector, and save it to a new array called multi_sample.
multi_sample = sapply(sample_sizes, function(x) genSamples(x, population))

Here's how we would do the same thing using a 'for' loop.

In [12]: 
%%R
# Create a new empty array to hold our new sample means.
loop_multi_sample = c()
# For every sample size in our list...
for (i in sample_sizes) {
    # Calculate statistics for a sample of the given size.
    loop_sample_stats = genSamples(i, population)
    # Add this mean value to our array by re-defining it as the existing array + the new vector of statistics.
    loop_multi_sample = c(loop_multi_sample, loop_sample_stats)
}
